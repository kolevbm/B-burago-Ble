Хардуер

Необходими компоненти

  • дву-осов джойстик с бутон
  • микроконтролер ESP32 DEV KIT DOIT
  • клипс за батерия 9V
  • превключвател (ключ)
  • подходяща кутия
  • Esp32-101-board
  • power bank

Схема на свързване

Захранваме контролера чрез батерия 9V на пинове Vin => +batt и GND => -batt, като прекъсваме плюса през ключ.

Джойстикът има 5 пина и се свързва по следния начин към контролера

Pin предназначение
GND маса
Vcc захранване с работното напрежение на контролера
X аналогова стойност по X
Y аналогова стойност по Y
Sel цифров сигнал от натискане на бутона

Joystick => Esp32

  • GND => GND
  • Vcc => 3V3
  • X => VN
  • Y=> VP
  • Sel => D32
